# 🎓 VarsityHub Portal

VarsityHub is a web-based university campus service management system.  
It allows students, teachers, and staff to book guest houses, conference rooms, and sports facilities through a secure role-based system.

🌐 Live Project:  
https://sweproject.gt.tc

---

## 📌 Project Overview

VarsityHub simplifies campus facility management by providing:

- Online booking system for university services
- Role-based dashboards
- Admin approval workflow
- Organized database management
- Secure login system

---

## 🚀 Features

### 🏨 Guest House Booking
- Book Single, Double, or Triple rooms
- Select available guest houses
- Track booking status (Pending / Approved / Rejected)

### 🏢 Conference Room Booking
- Submit conference room booking requests
- Digital approval system
- Manage reservation schedules

### 🏏 Sports Facilities Management
- Book sports facilities and equipment
- Request approval from PE Admin
- Organized sports management dashboard

### 🔐 Role-Based Access Control

Separate dashboards and permissions for:

- Student
- Teacher
- Staff
- System Admin
- Conference Admin
- PE Admin

Each role has specific features and access levels.

---

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)
